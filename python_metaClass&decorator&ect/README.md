# Python 元编程与装饰器学习项目

这是一个用于探索 Python 元编程和装饰器边界能力的学习项目。

## 项目结构

```
project/
├── main.py              # 项目入口
├── decorators/          # 装饰器模块 (4+3个)
├── metaclasses/        # 元类模块 (1个)
├── descriptors/       # 描述符模块 (1个)
└── advanced/         # 高级主题 (55个)
```

## 覆盖范围 (55章)

从装饰器基础到高级异步并发,从数据处理到函数式编程,覆盖Python核心知识体系.
